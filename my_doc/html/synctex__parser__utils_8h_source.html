109 /* If the core of the last path component of src is not already enclosed with double quotes ('"')
110  * and contains a space character (' '), then a new buffer is created, the src is copied and quotes are added.
111  * In all other cases, no destination buffer is created and the src is not copied.
112  * 0 on success, which means no error, something non 0 means error, mainly due to memory allocation failure, or bad parameter.
113  * This is used to fix a bug in the first version of pdftex with synctex (1.40.9) for which names with spaces
114  * were not managed in a standard way.
115  * On success, the caller owns the buffer pointed to by dest_ref (is any) and
116  * is responsible of freeing the memory when done.
117  * The size argument is the size of the src buffer. On return the dest_ref points to a buffer sized size+2.*/
118 int _synctex_copy_with_quoting_last_path_component(const char * src, char ** dest_ref, size_t size);
