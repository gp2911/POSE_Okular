Detailed Description

A toolbar that sets ThumbnailList properties when clicking on items.

This class is the small toolbar that resides in the bottom of the ThumbnailsBox container (below ThumbnailList and the SearchLine) and emits signals whenever a button is pressed. A click action results in invoking some method (or slot) in ThumbnailList.
