bool KTreeViewSearchLine::Private::checkItemParentsVisible ( QTreeView *  treeView,
const QModelIndex &  index 
)

Check whether item, its siblings and their descendents should be shown. Show or hide the items as necessary.

item The list view item to start showing / hiding items at. Typically, this is the first child of another item, or the the first child of the list view.

Returns
true if an item which should be visible is found, false if all items found should be hidden. If this function returns true and highestHiddenParent was not 0, highestHiddenParent will have been shown.
