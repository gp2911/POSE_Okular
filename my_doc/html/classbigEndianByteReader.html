Detailed Description

Byte reading routines which read big endian numbers from memory and convert them to native integers.

quint8* bigEndianByteReader::end_pointer

This pointer marks the end of the memory area where bytes can be read. It should point to the first byte which CANNOT be read. The idea is to have a safety net which protects us against SEGFAULTs. This is also used in virtual fonts, where the macro does not have an EOP command at the end of the macro.
