Detailed Description

A generator that builds contents from a PDF document.

All Generator features are supported and implented by this one. Internally this holds a reference to xpdf's core objects and provides contents generation using the PDFDoc object and a couple of OutputDevices called Okular::OutputDev and Okular::TextDev (both defined in gp_outputdev.h).

For generating page contents we tell PDFDoc to render a page and grab contents from out OutputDevs when rendering finishes.
