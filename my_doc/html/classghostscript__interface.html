QString ghostscript_interface::locateEPSfile ( const QString &  filename,
const KUrl &  base 
)
static

This method tries to find the PostScript file 'filename' in the DVI file's directory (if the base-URL indicates that the DVI file is local), and, if that fails, uses kpsewhich to find the file. If the file is found, the full path (including file name) is returned. Otherwise, the method returns the first argument. TODO: use the DVI file's baseURL, once this is implemented.
