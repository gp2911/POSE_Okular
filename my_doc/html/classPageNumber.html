Detailed Description

Class to represent a page number.

The class PageNumber is really nothing but an alias for quint16, and can be casted to and from quint16. It is used in kviewshell to remind the programmer of the convention that page numbers start at '1' (for 'first page'), and that the value '0' means 'illegal page number' or 'no page number'. Accordingly, the value '0' is also named PageNumber::invalidPage, and there is a trivial method isInvalid() that checks if the page number is 0.
